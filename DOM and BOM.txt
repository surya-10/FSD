Window object is supported by all browers. Global varibles are properties of window object. Global function are methods of the window object. Every document is a property of the window object.
Window is at a root level at the javascript object. It is root object of the Document Object Model. 
The Window interface is home to a variety of functions, namespaces, objects, and constructors which are not necessarily directly associated with the concept of a user interface window.

Document object represents your web page. 
A document is an object inside the window object and we use a document object for manipulation inside the document. The document contains the contents of the page.'  The Document object has various properties that refer to other objects which allow access to and modification of document content.
